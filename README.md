# 도라가DORAGA: 교통 분산 시스템

## 프로젝트 개요
- 개별 사용자들의 최단 경로 선택으로 인한 병목현상을 해결하기 위해 포인트 지급 등의 경제적 보상을 도입하여 사용자를 우회경로로 유도하여 쏠림현상을 해결하는 AI기반교통제어 프로젝트입니다.
- 실시간으로 경로를 분석하여 우회경로를 알려주고, @분 추가하고 @원 받기! 식으로 사용자를 우회경로로 유도하여 병목현상을 해결합니다.
- 택시조합, 운송기업 등과 같이 급한 차량들과 제휴를 맺어 예산을 해결하고 우회경로를 선택한 사용자들에게 포인트를 지급합니다.

## 1/12 ~ 2/6 동안의 프로젝트 계획
- 이번 프로젝트에서는 실제 모바일 앱 출시보다는 알고리즘의 유효성을 증명할 수 있는 시뮬레이션 웹 서비스 개발이 최종 목표입니다.

## 예상 사용 기술 스택
### 1. Language & Environment
- **Python 3.10+**: 메인 개발 언어
- **Virtualenv**: 독립적인 라이브러리 환경 관리

### 2. Traffic & Graph Analysis (Core)
- **OSMnx**: OpenStreetMap 기반 실제 도로망 데이터 추출 및 전처리
- **NetworkX**: 도로망 그래프화 및 최단 경로(Dijkstra) 알고리즘 구현
- **GeoPandas**: 지리 공간 데이터(GIS) 처리 및 연산

### 3. Data & Math Engine
- **Pandas / NumPy**: 시뮬레이션 로그 데이터 처리 및 BPR 수식 연산
- **Math**: 동적 리워드 산출 및 교통량-통행시간 함수 구현

### 4. Interactive Simulation UI
- **Streamlit**: 시뮬레이션 제어 및 결과를 보여주는 웹 대시보드 구축
- **Pydeck**: 도로망 정체 현황 및 차량 이동 실시간 시각화 (Map-based)
- **Plotly**: 정체 해소율 및 경제적 효용 분석 차트 시각화

### 5. Backend (Extensibility)
- **FastAPI**: 알고리즘 엔진의 API 서버화 (향후 서비스 확장 대비)
